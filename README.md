# O/M (OFF MENU) Design

A premium agency website design focused on brand identity and digital experiences. This design was crafted to showcase a minimalist, modern aesthetic with a focus on high-impact typography and clear calls to action.

![Design Preview](imgs/design.png)

##  Overview

The **O/M Design** represents a sophisticated digital presence for a creative agency. It emphasizes large, bold statements and a clean layout that builds trust and highlights professional expertise.

##  Key Features

- **Minimalist Hero Section**: A clean, spacious layout that makes the core message stand out.
- **Strong Value Proposition**: "Designing Brands and Experiences Worth Remembering".
- **Strategic CTAs**: 
  - `Get Started`
  - `View Pricing`
  - `Book a Call`
- **Trust Indicators**: A client showcase section featuring world-class brand associations like Tangent, Momentum, and Superintelligent.
- **Glassmorphism Elements**: Subtle use of blurred backgrounds and floating UI elements for a modern feel.

##  Tools Used

- **Figma**: For the UI/UX design and prototyping.
- **Adobe Illustrator**: For potential vector asset creation.
